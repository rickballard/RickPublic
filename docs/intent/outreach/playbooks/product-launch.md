# Product Launch Checklist (Template)

## Pre‑launch
- Positioning one‑pager
- Demo GIF (≤90s)
- README polish; quickstart; FAQ
- Issue templates for pilots
- Release notes & changelog

## Launch
- Substack post
- GitHub Release
- LinkedIn post + Discussion thread
- Email to target list

## Post‑launch
- Track pilot leads; calendar links
- Collect metrics; publish learnings
